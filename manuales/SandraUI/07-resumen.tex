\chapter*{Resumen}

Sandra UI Consola es la interfaz gráfica de administración del ecosistema Sandra, diseñada para proporcionar una experiencia de usuario moderna y eficiente para la gestión de sistemas middleware empresariales. Este manual describe en detalle todas las funcionalidades disponibles en la consola, desde los conceptos básicos de navegación hasta las capacidades avanzadas de configuración y monitoreo.

La consola está construida sobre tecnologías modernas como Angular con componentes standalone y signals reactivos, garantizando un rendimiento óptimo y una experiencia de usuario fluida. El sistema implementa robustos mecanismos de seguridad incluyendo autenticación JWT, autenticación multifactor (MFA) mediante TOTP, y un Web Application Firewall (WAF) avanzado.

A través de este manual, el lector aprenderá a utilizar Effectively todas las funcionalidades de Sandra UI Consola, incluyendo la gestión de usuarios y permisos, el monitoreo en tiempo real del sistema, la configuración de alertas, y la generación de reportes. El documento también aborda aspectos de cumplimiento normativo y mejores prácticas de seguridad alineadas con estándares internacionales como ISO 27001 y ISO 22301.

\textbf{Palabras clave:} Sandra, Consola, Administración, Middleware, Interfaz Gráfica, Angular, Seguridad, JWT, MFA, Observabilidad
