\section{Capturas de Pantalla}

Los siguientes anexos muestran las pantallas principales de Sandra UI Consola:

\subsection{Pantalla de Login}

La pantalla de inicio de sesión es el punto de acceso seguro a la consola. Incluye campos para usuario y contraseña, además de la opción de autenticación multifactor.

\subsection{Validación 2FA}

La validación de autenticación de dos factores mediante código TOTP de 6 dígitos.

\subsection{Cambio de Contraseña}

Interfaz para el cambio seguro de contraseña con validación de requisitos.

\subsection{Certificados SSL}

Gestión de certificados SSL para conexiones seguras.

\subsection{Validación de Certificados}

Herramienta para validar la integridad y vigencia de certificados.

\subsection{Workflow de Estados}

Diagrama que muestra los diferentes estados del sistema y las transiciones entre ellos.
