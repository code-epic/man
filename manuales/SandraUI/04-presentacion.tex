\chapter*{Presentación}

Sandra UI Consola se posiciona como una solución Enterprise de alta gama, integrando capacidades de SRE y observabilidad avanzada para garantizar la continuidad operativa. La consola proporciona una experiencia de usuario premium basada en principios de diseño moderno, mientras mantiene los más altos estándares de seguridad y rendimiento.

\section*{Objetivo del Manual}

Este manual tiene como objetivo proporcionar una guía completa para el uso de Sandra UI Consola. A través de sus capítulos, el lector conocerá desde los conceptos básicos hasta las funcionalidades avanzadas de configuración y monitoreo.

El manual está estructurado para servir tanto como referencia rápida como guía de aprendizaje, permitiendo a usuarios de diferentes niveles de experiencia aprovechar al máximo las capacidades de la consola.

\section*{Audiencia Objetivo}

Este documento está dirigido a:

\begin{itemize}
    \item Administradores de sistemas que gestionan la infraestructura Sandra
    \item Operadores que monitorean el estado del ecosistema
    \item Desarrolladores que integran aplicaciones con Sandra
    \item Personal de seguridad que supervisa el cumplimiento normativo
\end{itemize}
